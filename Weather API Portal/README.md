
# 🌤 Weather API Portal

A modern real-time weather web application built using HTML, CSS, and JavaScript.  
It integrates the OpenWeatherMap API to deliver live weather data based on user input or geolocation.

---

## 🚀 Live Features

- 🔍 Search weather by city name
- 📍 Get weather using device location (Geolocation API)
- 🌡 Real-time temperature updates
- 💧 Humidity, wind speed, pressure, and "feels like" data
- 🌤 Dynamic weather icons based on conditions
- ⚡ Async/Await API handling for smooth data fetching
- 📱 Fully responsive design for mobile and desktop
- 🔄 Auto-updating UI without page reload

## ⚙️ How It Works

1. User enters a city name OR clicks "Use My Location"
2. App sends request to OpenWeatherMap API
3. API returns live weather data in JSON format
4. JavaScript processes and displays data dynamically on UI
